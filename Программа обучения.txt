Программа обучения SQL для магистров (12 занятий по 1.5 часа)

План занятий:

Занятие 1: Введение в базы данных и SQL (03.09.2025)
Теория:
- Что такое база данных (БД)
- Системы управления базами данных (СУБД)
- Виды БД: реляционные vs нереляционные
- Обзор реляционных СУБД: OLTP vs OLAP

Занятие 2: Основы SQL. Простые SELECT-запросы (03.09.2025)
Теория:
- Структура SQL-запроса
- Основные команды: SELECT, FROM, WHERE
- Сортировка (ORDER BY)
- Ограничение количества строк (LIMIT, OFFSET)

Занятие 3: Типы данных (10.09.2025)
Теория:
- Типы данных

Занятие 4: Агрегатные функции и группировка (10.09.2025)
Теория:
- Функции: COUNT, SUM, AVG, MIN, MAX
- Группировка (GROUP BY)
- Фильтрация групп (HAVING)

Занятие 5: Подзапросы
Теория:
- Вложенные запросы
- Подзапросы в WHERE, FROM, SELECT

Занятие 6: Соединение таблиц (JOINs)
Теория:
- Виды соединений: INNER, LEFT/RIGHT, FULL, CROSS JOIN
- Алиасы таблиц

Занятие 7: Временные таблицы и CTE
Теория:
- Временные таблицы
- CTE (WITH ... AS)
- Разница и применение

Занятие 8: Оконные функции
Теория:
- Функции: ROW_NUMBER(), RANK(), DENSE_RANK(), LEAD(), LAG()
- PARTITION BY, ORDER BY в окнах

Занятие 9: Индексы и их оптимизация
Теория:
- Что такое индексы
- Типы индексов (B-tree, Hash, GiST, GIN)
- Когда использовать индексы

Занятие 10: Оптимизация запросов
Теория:
- План выполнения запроса (EXPLAIN)
- Оптимизация JOIN, подзапросов, индексов

Занятие 11: Транзакции и ACID. Процедуры и функции в PostgreSQL.
Теория:
- Транзакции (BEGIN, COMMIT, ROLLBACK)
- Уровни изоляции
- Процедуры и функции в PostgreSQL
